How it works

4단계로 끝

설치부터 사용까지 복잡한 설정 없이 바로 시작하세요.

01
다운로드 & 실행
설치 파일을 받아 실행하면 깔끔한 다크 UI가 바로 열립니다.
02
API 키 입력
DeepL 또는 Gemini API 키를 설정 탭에 입력. AES-256으로 암호화 저장됩니다.
03
파일 선택
원하는 탭을 선택하고 파일을 불러옵니다.
04
결과 확인

FAQ

자주 묻는 질문

궁금한 점을 먼저 확인해보세요.

API 키 없이도 쓸 수 있는 기능이 있나요?+

네, 22개 기능 중 15개는 API 키 없이 완전 무료로 사용할 수 있어요.

API 키 불필요 (로컬 처리): 폴더 정리, 파일명 변경, 과제 폴더 생성, PDF 처리 5종, 이미지 일괄처리·변환·추출, 워터마크, 배경 제거, 엑셀 자동화, 과제 트래커

API 키 필요 (인터넷 사용): 파일명 번역(DeepL), 이미지 OCR·문서 요약·문서 초안·참고문헌 정리(Gemini)

API 키가 외부로 유출될 가능성이 있나요?+

없습니다. API 키는 이 PC에서만 암호화되어 로컬에 저장됩니다. Filelo 서버는 존재하지 않으며, 어떤 데이터도 외부로 전송되지 않습니다.

암호화 방식: AES-256-GCM + PBKDF2-SHA256 (480,000회) + 머신 고유 ID 기반 키 파생. 파일을 다른 PC로 복사해도 복호화가 불가능합니다.

어떤 파일 형식을 지원하나요?+

번역: .docx, .pdf, .txt
이미지: .jpg, .jpeg, .png, .webp, .bmp, .tiff
Excel: .xlsx, .xls, .csv
AI 요약·초안: .pdf, .txt, .docx
PDF 처리: .pdf
메타데이터 삭제: .pdf, .docx

인터넷 없이도 사용할 수 있나요?+

오프라인 완전 가능: 이미지 배경 제거(rembg), Excel 자동화, 과제 트래커, 폴더 정리, 파일명 변경, PDF 처리, 이미지 처리 등 대부분의 기능

인터넷 필요: DeepL 번역(파일명 번역), Gemini AI 기능(OCR·요약·초안·참고문헌)

rembg 배경 제거 모델은 첫 실행 시 한 번만 다운로드하면 이후 완전 오프라인 사용 가능합니다.

처리 결과는 어디에 저장되나요?+

처리 결과물은 원본 파일과 같은 폴더에 자동 저장됩니다. 원본은 절대 덮어쓰지 않아요.

보고서.docx → 보고서_EN.docx
data.xlsx → data_중복제거.xlsx
photo.jpg → _배경제거/photo_nobg.png
사진들 → _처리완료/ 폴더

무료인가요? 사용 제한이 있나요?+

Filelo 자체는 완전 무료입니다. 외부 API 서비스의 무료 한도만 참고하세요.

• DeepL Free: 500,000자/월 무료
• Gemini API: Google AI Studio 기준 무료 티어 제공
• 그 외 15개 기능: 완전 무료 · 무제한

여러 파일을 한 번에 처리할 수 있나요?+

네, 대부분의 기능이 일괄(배치) 처리를 지원합니다.

파일 추가 버튼으로 여러 개를 선택하거나, 폴더 선택으로 폴더 전체를 한 번에 처리할 수 있어요. 처리 중에는 진행 상황이 로그창에 실시간으로 표시됩니다.

웹 AI 도구에서 입력한 내용은 저장되나요?+

이 웹사이트의 AI 도구는 브라우저에서 직접 Gemini API를 호출합니다. 입력한 텍스트와 API 키는 Filelo 서버를 전혀 거치지 않으며, 브라우저를 닫으면 모두 사라집니다.

API 키는 sessionStorage에만 임시 저장되며 탭을 닫으면 즉시 삭제됩니다.

개인정보처리방침

1. 수집하는 개인정보

Filelo 웹사이트는 어떠한 개인정보도 수집하지 않습니다. 이 웹사이트는 정적 페이지로 운영되며, 서버로 전송되는 데이터가 없습니다.

2. API 키 처리

웹사이트의 AI 도구 체험 기능에 입력하는 Gemini API 키는 브라우저의 sessionStorage에만 임시 저장되며, 탭을 닫으면 즉시 삭제됩니다. 해당 키는 Filelo 서버를 전혀 거치지 않고 브라우저에서 Google Gemini API로 직접 전송됩니다.

3. 데스크톱 앱의 데이터 처리

Filelo 데스크톱 앱은 모든 데이터를 사용자의 PC에서만 처리합니다. API 키는 AES-256-GCM으로 암호화되어 로컬에 저장되며, 외부 서버로 전송되지 않습니다.

4. 쿠키 및 추적

이 웹사이트는 쿠키, 광고 추적, 분석 도구를 일절 사용하지 않습니다.

5. 문의

  • 소프트웨어를 무단으로 재배포하거나 상업적으로 판매하는 행위는 금지됩니다.
  • 소프트웨어를 역공학(리버스 엔지니어링)하거나 악의적 목적으로 사용하는 것을 금지합니다.

3. 면책 조항

Filelo는 "있는 그대로(AS-IS)" 제공됩니다. 소프트웨어 사용으로 인한 데이터 손실, 파일 손상 등의 피해에 대해 개발자는 책임을 지지 않습니다. 중요한 파일은 반드시 사전에 백업하세요.

4. 외부 API 이용

DeepL, Google Gemini 등 외부 API를 사용하는 기능은 각 서비스의 이용약관을 따릅니다. API 키 발급 및 사용에 따른 비용은 사용자 본인이 부담합니다.
